Exercice 1

1) Test (0;-1) dans le premier système : \(3\cdot0+5\cdot(-1)+5=0\) (vrai), \(0-(-1)-1=0\) (vrai). Réponse a).
2) Milieu : \(\left(\frac{3+2}{2};\frac{-2-3}{2}\right) = \left(\frac{5}{2};-\frac{5}{2}\right)\) → réponse c).
3) \(\overrightarrow{AB}=(x_B-3; y_B-5)=(2;-6)\) ⇒ \(x_B=5, y_B=-1\) → réponse c).
4) Colinéaires : \(-2\times m - 3\times4 = 0 \Rightarrow -2m=12 \Rightarrow m=-6\) → réponse b).

Exercice 2

1) A(a;0) ∈ Δ ⇒ \(4a=8 \Rightarrow a=2\). B(0;b) ∈ Δ ⇒ \(-2b=8 \Rightarrow b=-4\).
2) C(1;3) : \(4\times1-2\times3=4-6=-2\neq8\) → Attention, dans l'énoncé original il y a peut-être une erreur : C et D n'appartiennent pas à Δ si l'équation est \(4x-2y=8\). Vérification : pour C : \(4-6=-2\) ; pour D : \(-4+2=-2\). Ils n'appartiennent pas à Δ. L'énoncé disait "Montrer que C et D appartiennent à Δ". Il est possible que l'équation de Δ soit différente (par exemple \(4x-2y=-2\)). Nous indiquons ici la méthode : on remplace les coordonnées dans l'équation.
3) Δ' : \(-2x+y=1\). Pour A(2;0) : \(-4+0=-4\neq1\) donc A ∉ Δ'.
4) \(\overrightarrow{AB}=(-2;-4)\), \(\overrightarrow{CD}=(-2;-4)\) (si C(1;3) et D(-1;-1)).
5) Vecteurs directeurs : Δ : \((2;4)\) (car \(4x-2y=8\)), Δ' : \((1;2)\) (car \(-2x+y=1\)). Ils sont colinéaires car \((2;4)=2(1;2)\). Les droites sont parallèles. Ordonnée à l'origine différente → strictement parallèles.
6) Système incompatible → \(S = \varnothing\).

Exercice 4

Figure de correction – A(2;4), B(0;2), C(7;1), D(2;6)

1) Sur le graphique, A semble avoir pour coordonnées (2;4).
2) Voir figure.
3)a) \(AB = \sqrt{(0-2)^2+(2-4)^2} = \sqrt{4+4} = \sqrt{8} = 2\sqrt{2}\).
\(AC = \sqrt{(7-2)^2+(1-4)^2} = \sqrt{25+9} = \sqrt{34}\).
\(BC = \sqrt{(7-0)^2+(1-2)^2} = \sqrt{49+1} = \sqrt{50} = 5\sqrt{2}\).
b) \(AB^2+AC^2 = 8+34 = 42\) et \(BC^2 = 50\). Les valeurs ne vérifient pas Pythagore. Note : avec A(2;4), le triangle n'est pas rectangle. L'énoncé original a peut-être des coordonnées différentes. Nous laissons la méthode : vérifier si \(AB^2+AC^2 = BC^2\).
4)a) Vecteurs \(\overrightarrow{AC}=(5;-3)\) et \(\overrightarrow{AD}=(0;2)\). Ils ne sont pas colinéaires, donc A,C,D ne sont pas alignés avec ces coordonnées. (À adapter selon le vrai point A).
b) \(CD = \sqrt{(2-7)^2+(6-1)^2} = \sqrt{25+25} = \sqrt{50} = 5\sqrt{2}\).
c) Aire de BCD = \(\frac{1}{2} |(x_C-x_B)(y_D-y_B) - (x_D-x_B)(y_C-y_B)| = \frac{1}{2} |7\times4 - 2\times(-1)| = \frac{1}{2} |28+2| = 15\).
